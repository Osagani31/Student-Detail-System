# Student Details System

A simple web application built using **Java JSP and Servlets** to store and manage student details, including their name, degree, and photo upload. This mini project demonstrates how to handle file uploads (student photos) and persist data using MySQL via PHPMyAdmin, running on Apache Tomcat server and developed in Apache NetBeans IDE.

---

## Features

| Feature                         | Description                                                 |
|--------------------------------|-------------------------------------------------------------|
| Student Data Entry              | Add student details like name and degree                     |
| File Upload                    | Upload and save student photo securely on the server         |
| Data Persistence               | Store student info and photo path in MySQL database          |
| Dynamic Display                | View stored student details along with uploaded photos       |
| JSP & Servlet Integration       | Server-side logic with JSP frontend for data interaction      |
| Database Management            | MySQL database management via PHPMyAdmin                      |
| Built with                     | Java, JSP, Servlets, Apache Tomcat, MySQL, PHPMyAdmin        |

---

## How to Run

1. Import the project into **Apache NetBeans**.
2. Configure database connection in your Servlet (`db.properties` or connection class).
3. Run the MySQL server and set up your database and table (SQL script included).
4. Deploy the project on **Apache Tomcat** server.
5. Access the application on `http://localhost:8080/your-app-context`.
